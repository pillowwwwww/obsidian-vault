1. 什么是“前向推理”？

在神经网络中，“前向推理”（forward pass / inference）指的是：

- 给定一组输入样本（这里指公共数据集中的某张图像、文本或图文对），
- 将它们输入到已经训练好的模型（客户端本地模型），
- 通过模型的各层网络计算，得到输出（例如分类 logits、特征向量、或者隐藏层表示等）。

在联邦学习场景下，客户端本地模型是事先在它的私有数据上进行过训练（或部分训练）的，故模型参数中已经“编码”了客户端对其私有数据的学习结果。虽然公共数据与私有数据并不相同，但模型仍然可以对公共数据进行推理，输出表示向量。

#### NN和CNN的关系

- 神经网络是一个总称，指的是由神经元连接组成的模型。CNN 是为处理图像数据而特别设计的神经网络结构，能更有效捕捉空间信息。
- | 网络类型            | 全称                             | 简要说明                 | 应用场景             |
| --------------- | ------------------------------ | -------------------- | ---------------- |
| **MLP**         | Multilayer Perceptron          | 最基础的前馈神经网络，全连接层构成    | 数值预测、分类任务        |
| **RNN**         | Recurrent Neural Network       | 具有时间记忆结构，能处理序列数据     | 语音识别、文本生成、时间序列预测 |
| **LSTM**        | Long Short-Term Memory         | RNN 的改进型，解决长距离依赖问题   | 机器翻译、对话系统        |
| **GRU**         | Gated Recurrent Unit           | LSTM 的简化版本           | 与 LSTM 类似，但参数更少  |
| **CNN**         | Convolutional Neural Network   | 使用卷积提取局部特征，适合图像类任务   | 图像识别、医学图像、目标检测   |
| **GAN**         | Generative Adversarial Network | 由生成器和判别器组成的对抗网络      | 图像生成、风格迁移、图像修复   |
| **Transformer** | ——                             | 基于注意力机制的架构，支持大规模并行计算 | 自然语言处理（如ChatGPT） |
| **GNN**         | Graph Neural Network           | 用于处理图结构数据的神经网络       | 社交网络、知识图谱、药物发现   |
