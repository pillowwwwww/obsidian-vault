### 2. CUDA torch版本安装
各位开发者们，查找实际的安装信息真是太麻烦了。Anaconda、CUDA Toolkit、cuDNN、环境变量，还要编译什么东西？？
其实只需要以下几步|：
- 1.
`nvidia-smi`
可以查看CUDA版本，我的电脑支持CUDA 版本：12.7
你的驱动虽然支持 CUDA 12.7，但 PyTorch 并没有 cu127 对应的安装包。
所以你仍需手动安装 CUDA 11.8 对应的 PyTorch 版本（cu118），它在 CUDA 向后兼容机制下 是可以在 CUDA 12.7 驱动上运行的。
- 2.
```
pip3 install --pre torch torchvision torchaudio --index-url https://download.pytorch.org/whl/nightly/cu126
```
- 3. 并且命令“torch.cuda.is_available()”
