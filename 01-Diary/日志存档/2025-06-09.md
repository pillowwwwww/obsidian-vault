## 📊 训练日志常见指标解释

| 指标名 | 全称/含义 | 解释 |
|--------|------------|------|
| Train Loss | 训练损失 | 联邦平均后的模型在客户端训练数据上的平均损失值 |
| Test Accuracy | 测试准确率 | 聚合模型在全局测试集上的准确率（分类正确的比例） |
| Test AUC | Area Under ROC Curve | ROC曲线下的面积，用于衡量分类器排序能力（越接近 1 越好） |
| Std Test Accuracy | Accuracy 的标准差 | 表示各客户端模型准确率的离散程度 |
| Std Test AUC | AUC 的标准差 | 表示各客户端模型 AUC 的差异情况 |

## ✅ 各类指标意义详解

### 🎯 1. Train Loss（训练损失）
- 趋势应逐渐 **下降**。
- 如果 loss 长时间维持不变或波动，说明模型 **训练未收敛或过早收敛**。

### 🧪 2. Test Accuracy（测试准确率）
- 表征模型的 **泛化能力**。
- 应随着训练轮数逐步 **上升**，最终趋于平稳。

### 📈 3. AUC（Area Under Curve）
- 衡量模型对正负样本的区分能力。
- 比准确率更稳定，特别在类不平衡时更有用。
- AUC ≈ 0.5 表示随机猜测，AUC → 1.0 表示优秀分类。

### 📉 4. 标准差（Std）
- 说明客户端之间的性能差异。
- Std 越小，说明 **各客户端模型一致性越好**。
- Std 较大则可能表明 **数据异构性高或训练效果不平衡**。

## ❓ 拟合状态如何判断？

| 状态 | 判断依据 | 特征表现 |
|------|-----------|----------|
| 欠拟合 | Train Loss 高，Test Accuracy 低且不变 | 模型没有学会数据，容量不足或训练不足 |
| 过拟合 | Train Loss 很低，Test Accuracy 先上升后下降 | 模型记住了训练集，泛化能力差 |
| 拟合饱和 | Train Loss 和 Test Accuracy 均趋于稳定，不再明显变化 | 模型已收敛，再训练也提升有限 |
